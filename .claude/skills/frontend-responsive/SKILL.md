# Frontend Responsive

## When to use this skill:

- When implementing mobile-first layouts
- When writing media queries for different breakpoints
- When using responsive utility classes (Tailwind sm:, md:, lg:, xl:)
- When creating fluid, percentage-based layouts
- When using relative units (rem, em) instead of fixed pixels
- When optimizing images for different screen sizes
- When implementing responsive typography
- When designing touch-friendly tap targets (minimum 44x44px)
- When testing UI across mobile, tablet, and desktop viewports
- When prioritizing content visibility on smaller screens
- When implementing responsive navigation patterns
- When optimizing performance for mobile networks
- When reviewing layouts for consistent cross-device experience
- When working on grid or flexbox layouts that adapt to screen size

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.
